Here you can see all the Facts that are in your Gedcom file.

There are filters so that you can search each type of ancestor separately, or in any combination.

There is a also a surname filter search box, which will then just show those surnames.

Its worth remembering that the 5 Relationship Type filters also work when searching by surname.

When you have the report open, you can sort each column.

If you sort on the "Sources" column, this will put all the facts with no sources at the bottom enabling you to use the list to find the sources that still need to be added.
