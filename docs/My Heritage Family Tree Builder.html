To add the Lost Cousins event:

  • Double click on the person involved.
  • Select the Facts tab.
  • Click on the New Fact button.
  • In the enter custom type window type Lost Cousins
  • Click OK
  • In the Date window type the year of the census (1841 or 1881 or 1911)
  • Click OK

For future use the Lost Cousins event will be available from the enter custom type drop down list.

To export a GEDCOM file:

  • Update your findings
  • Click on File at the left of the top toolbar
  • Select Export GEDCOM from the drop down list
  • Click on Next
  • Navigate to the folder in which you wish to save the file
  • Name the file in the File name window
  • Click Save
  • If the file alfeady exists you will be asked if you want to replace it - click Yes
  • Click Next
  • Click Next
  • Modify the list of options if required
  • Click Next
  • It is not necessary for this task to save a folder of photos so click No
  • The GEDCOM file will be created - click Finish to complete the process.

To ensure that the Individual ID numbers correspond between the program and FTA:

  • From the toolbar click on Tools and then select Options
  • Select General
  • Scroll down to the entry Show IDs of elements in your Project
  • Click on the entry in the right hand column
  • Select Show Consecutive IDs
  • Click on OK
  • Restart the program
