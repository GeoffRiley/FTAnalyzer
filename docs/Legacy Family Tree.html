To create the Lost Cousins event:

  • On the tool bar click on View and select Master Lists from the drop down list and then Event Definition from the further options.
  • Click on the Add button
  • Type Lost Cousins in to the window and click on the Save button.
  • Click on the Close button

To add the event to a specific person:

  • Double click on the person to open the Individual’s Information window (alternatively if you are in the Family tab click on the Show Events for this Person button)
  • Next to the list of Events/Facts click on the Add button
  • Click on the inverted triangle to the right of the Event/Fact window at the top
  • Scroll down the list and select the Lost Cousins entry
  • Click on the Select button
  • Enter the census year in the Date window and click on the Save button
  • This will return you to the Individual’s Information window
  • It is possible to sort the list of events by date by clicking on the Options button and selecting that option
  • Click on the Save button

 

To create the Children Status event:

  • On the tool bar click on View and select Master Lists from the drop down list and then Event Definition from the further options.
  • Click on the Add button
  • Type Children Status in to the window and click on the Save button.
  • Click on the Close button

To add the event to a couple:

  • Double click on the couple's Marriage bar to open the Marriage/Relationship Information window (alternatively if you are in the Family tab click on the Marriage Events button)
  • Next to the list of Events/Facts click on the Add button
  • Click on the inverted triangle to the right of the Event/Fact window at the top
  • Scroll down the list and select the Children Status entry
  • Click on the Select button
  • Enter into the description using this format, 8 Total, 7 Alive, 1 Dead
  • Enter the census year in the Date window and click on the Save button
  • This will return you to the Family Information window

 

To export a GEDCOM file:

  • On the tool bar click on File and select Export to from the drop down list and then GEDCOM File from the further options (alternatively on your keyboard press Ctrl and E at the same time)
  • In the Produce file for window click on the inverted triangle and select the GEDCOM 5.5.1.Only option from the drop down list
  • In the Character Set window select the ANSI option from the drop down list
  • Click on the Record Selection button if you want to restrict the contents of the GEDCOM but normally this will be left at All Records in the Entire Family Tree
  • Click on the large button labelled Select File Name and START EXPORT
  • Navigate to the folder you wish to save your GEDCOM file to and enter your preferred name for the file in the File name window
  • Click on the Save button
  • You may add your details to the Compiler window or not
  • Click on the OK button
  • If you have left the Compiler window empty  click on No in the following window
  • After receiving confirmation of the export click on OK
